## When to use
Use this skill whenever the user asks to:
- tweak frontend
- update UI
- modify styling
- adjust layout
- change frontend behavior

## Post-Completion Behavior

After completing the requested frontend changes:
1. DO NOT run the app.
   Assume the user already has the app running locally.
2. DO NOT build the project unless explicitly asked.
3. The user will manually verify the changes.
4. If errors occur, the user will report them for fixing.
